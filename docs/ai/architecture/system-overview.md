# System Overview

## Product Definition
The product is a modular AI career growth system with five main modules:
1. AI Assistant
2. Daily Warmup
3. Interview
4. Coach
5. Negotiation

These modules share one AI Core but must remain independently usable.

## Core Product Rule
Each module must provide standalone value while optionally contributing signals, recommendations, and progress data to the wider system.

## Shared AI Core
The shared AI Core supports:
- intent detection,
- role and seniority adaptation,
- constructive feedback generation,
- multimodal interpretation,
- skills inference,
- skills verification,
- report generation,
- recommendation support.

## Modules Are Complementary, Not Mandatory
Users may start in any module and stop after any module.

Valid journeys include:
- Assistant only
- Warmup only
- Interview only
- Coach only
- Negotiation only
- Any optional combination of the above

No module should require completion of another module before access is granted.
