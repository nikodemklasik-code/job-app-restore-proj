# coach-engine

## Purpose
The coach-engine owns targeted interview skill development.

## Responsibilities
- answer analysis
- stronger rewritten answers
- STAR repair
- measurable impact repair
- ownership repair
- training plans
- targeted practice modules

## Must Do
- evaluate the answer, not the person,
- use evidence from the answer,
- stay constructive,
- help the user strengthen specific areas.

## Must Not Do
- behave like a full interviewer,
- produce hiring verdicts,
- override module independence rules.
