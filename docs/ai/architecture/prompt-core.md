# prompt-core

## Purpose
prompt-core contains shared AI rules across the system.

## Responsibilities
- system core rules
- tone rules
- constructive feedback rules
- compliance rules
- multimodal rules
- adaptation rules
- output format rules

## Should Not Contain
- persona-specific rules
- interview-only prompts
- coach-only prompts
- report payload mapping
- session state logic

## Design Rule
Changes here are high-impact and must stay aligned with documented product rules.
