# Module Independence Rule

## Core Rule
All product modules are complementary but independently usable.

## Meaning
- Assistant does not require Interview.
- Warmup does not require Coach.
- Coach does not require Interview.
- Negotiation does not require Warmup.
- Skills verification does not require a fixed funnel.

## Allowed Cross-Module Behavior
- optional recommendations,
- optional handoffs,
- optional use of shared context,
- optional reuse of observed signals.

## Forbidden Cross-Module Behavior
- mandatory Interview IDs for Coach,
- locked modules behind completion requirements,
- forced onboarding funnel through all modules,
- hidden hard dependencies disguised as recommendations.

## Engineering Rule
Use optional context injection, not hard gating.
