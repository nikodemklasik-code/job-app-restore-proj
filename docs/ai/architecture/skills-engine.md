# skills-engine

## Purpose
skills-engine tracks declared, observed, strengthening, verified, and strong-signal skills.

## Responsibilities
- skill evidence tracking
- skill state transitions
- skill verification logic
- source-aware skill history
- progress signal generation

## Skill States
- Declared
- Observed
- Strengthening
- Verified
- Strong Signal

## Rule
A skill should not become strong from one isolated answer.
Repeated evidence or explicit verification is required.
