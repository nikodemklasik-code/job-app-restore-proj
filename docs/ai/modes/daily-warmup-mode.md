# Daily Warmup Mode

## Purpose
Daily Warmup is a lightweight daily speaking drill.

## Key Rule
Warmup is a micro-practice ritual, not a full interview and not a full coaching report.

## Expected Elements
- one prepared question,
- readable question,
- optional read-aloud question,
- timed answer,
- voice answer,
- short evaluation,
- optional retry.

## Independence Rule
Warmup must provide full value even when used alone.
