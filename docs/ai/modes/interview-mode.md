# Interview Mode

## Purpose
Interview is the full realistic mock interview experience.

## Key Rule
Interview evaluates the user's overall interview performance across multiple signals.

## Must Include
- adaptive questioning,
- realistic pacing,
- role calibration,
- spoken closing summary,
- detailed written report,
- optional coach handoff.

## Independence Rule
Interview must stand on its own even if the user has never used Warmup or Coach.
