# AI Career AI Master Bundle

This package contains the consolidated AI documentation and prompt rules for the AI career system.

It includes:

- product specs,
- module definitions,
- KPI documents,
- coach and negotiation behavior specs,
- principles for capacity adaptation, skill growth, neurodiversity-aware coaching,
- adversarial and abuse resistance,
- AI roadmap and evaluation dataset spec,
- shared prompt rule files for the backend (`backend/src/prompts/shared/`).

This package is intended as a repo-ready documentation and AI behavior layer.

## Where to start

- **Human-readable docs (this tree):** [docs/ai/README.md](./README.md)
- **Executable prompt strings (backend):** `backend/src/prompts/` — barrel file `index.ts` re-exports shared rule modules.
