# KPIs and Success Metrics

## Purpose

These KPIs exist to measure:

- whether the modules create real value,
- whether users return,
- whether they improve,
- whether the AI is trusted,
- whether the system is healthy enough to scale.

---

## System-Level KPIs

### Activation

Measures whether users reach meaningful value quickly.

Core metrics:

- meaningful AI session completion rate
- time to first useful outcome
- first-week multi-module usage rate

### Retention

Measures whether users return.

Core metrics:

- D1 retention of AI-active users
- D7 retention of AI-active users
- D30 retention of AI-active users

### Progress

Measures whether users improve.

Core metrics:

- 30-day measurable improvement rate
- retry improvement rate
- repeated weakness reduction rate
- skill state upgrade rate

### Trust and Quality

Measures whether the AI is useful and credible.

Core metrics:

- AI satisfaction score
- AI trust score
- report usefulness score
- hallucination risk rate
- unsafe output rate

### Monetization

Measures whether value turns into revenue.

Core metrics:

- free to paid conversion
- conversion by first-entry module
- paid retention
- premium usage rates by module

---

## Module KPI Framework

Every module must have:

- one usage KPI,
- one completion KPI,
- one usefulness KPI,
- one repeat or progression KPI.

---

## Assistant KPIs

- Usage: question submission rate
- Completion: resolved session rate
- Usefulness: answer satisfaction score
- Repeat/Contribution: CTA click-through into deeper modules

## Daily Warmup KPIs

- Usage: daily participation rate
- Completion: warmup completion rate
- Usefulness: warmup usefulness score
- Repeat/Contribution: retry improvement rate and streak continuation rate

## Interview KPIs

- Usage: interview start rate
- Completion: interview completion rate
- Usefulness: perceived realism/usefulness score
- Repeat/Contribution: report open rate, PDF download rate, Coach handoff rate

## Coach KPIs

- Usage: coach module start rate
- Completion: coach module completion rate
- Usefulness: coach usefulness score
- Repeat/Contribution: retry improvement rate and return rate

## Skill Lab KPIs

- Usage: skill verification entry rate
- Completion: verification completion rate
- Usefulness: skill trust score
- Repeat/Contribution: Observed -> Verified transition rate

## Job Discovery KPIs

- Usage: job search usage rate
- Completion: explain-fit view completion
- Usefulness: relevance / fit usefulness score
- Repeat/Contribution: apply click-through rate

## Job Radar KPIs

- Usage: radar open rate
- Completion: benchmark/comparison section completion
- Usefulness: employer/offer intelligence usefulness score
- Repeat/Contribution: usage before apply or follow-up

## Application Review KPIs

- Usage: review open rate
- Completion: next-step recommendation generation rate
- Usefulness: clarity after review score
- Repeat/Contribution: follow-up action click rate

## Document Lab KPIs

- Usage: document upload rate
- Completion: upload and extraction success rate
- Usefulness: downstream usage of extracted context
- Repeat/Contribution: profile enrichment rate

## Negotiation KPIs

- Usage: strategy mode usage rate
- Completion: session-to-final-output rate
- Usefulness: negotiation usefulness score and confidence lift
- Repeat/Contribution: message export rate and practice completion rate

---

## AI Quality KPIs

### Quality

- relevance score
- specificity score
- constructiveness score
- grounding score
- hallucination risk rate
- harmful output rate

### Reliability

- latency p50
- latency p95
- fail rate
- retry rate
- partial response break rate
- SSE interruption rate
- STT success rate
- TTS success rate

### Safety

- policy pass rate
- abuse-resistance pass rate
- jailbreak resistance pass rate
- invalid-signal suppression rate
- unsafe recommendation rate
