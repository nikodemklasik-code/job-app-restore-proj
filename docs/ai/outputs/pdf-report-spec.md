# PDF Report Spec

## Cel
Pełna analiza rozmowy po zakończeniu sesji.

## Sekcje PDF
1. Session Metadata
2. Overall Summary
3. Top Strengths
4. Areas to Strengthen
5. Recruiter Perspective
6. Selected Answer Analysis
7. Recommended Next Interview Focus
8. Coach Handoff Recommendations

## Tone
- konstruktywny,
- precyzyjny,
- oparty na obserwacjach,
- bez języka obniżającego.

## Zasada
PDF ma rozwijać spoken summary, nie powielać go 1:1.
