# MultivoHub — values and philosophy (design inspiration)

This folder holds **canonical inspiration decks** for product tone, UX framing, and brand values. They are **not** runtime specifications; use them alongside `docs/features/product-screens-spec-v1.0.md` and PO/QC task cards.

## Files

| File | Language | Role |
|------|----------|------|
| `Multivohub_katalog_wartosci_i_filozofii.pptx` | PL | Catalogue of values and philosophy (source narrative). |
| `Multivohub_values_philosophy_EN_edited.pptx` | EN | Edited English deck for EN-first product copy and international alignment. |

## How to use in the repo

- **Copy / IA / UI:** align module purpose, CTA tone, and “emotional effect” lines with these decks before shipping major surfaces.
- **Assistant / Coach / Analysis:** check that guidance language matches stated philosophy (agency, evidence, no over-claiming).
- **QC:** may treat divergence from agreed values language as a product consistency issue (per PO).
