# Remaining Screens — Production Ready Pack v1

Ten pakiet domyka **pozostałe screeny produktowe** względem pakietu „8 modułów” ([`8-modules-production-ready-pack-v1.md`](./8-modules-production-ready-pack-v1.md)), który obejmuje m.in.:
- Legal Hub
- Skill Lab
- Case Practice
- Interview
- Coach
- Daily Warmup
- Negotiation
- Job Search

## Ten pakiet obejmuje
- Dashboard
- Profile
- Applications
- Applications Review
- Document Lab
- Style Studio
- AI Assistant
- AI Analysis
- Job Radar
- Community Centre
- Settings
- Billing

## Zawartość
- osobny production-ready spec dla każdego brakującego screenu
- first production slices TSV
- RFQ template
- READY_FOR_QC template

## Zasada użycia
Najpierw wybierz bounded slice z TSV, potem skopiuj RFQ template, utwórz fizyczny REPORT_PATH w docs/qc-reports, dopisz do chaina i dopiero wtedy odpal health/loop.
