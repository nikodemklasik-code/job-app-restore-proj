# Agent work — RFQ drafts

`docs/squad/19-SCREENS_FIRST_PRODUCTION_SLICES.tsv` names **target** RFQ paths under `docs/agent-work/` (e.g. `agent-1-profile-completeness-rfq.md`). The **Dashboard** first slice uses the same filenames as [`../squad/REMAINING-SCREENS_FIRST_PRODUCTION_SLICES.tsv`](../squad/REMAINING-SCREENS_FIRST_PRODUCTION_SLICES.tsv): `dashboard-dashboard-aggregate-snapshot-rfq.md` / `dashboard-dashboard-aggregate-snapshot-ready-for-qc.md`. Paths are targets until created from [`../squad/19-SCREENS_RFQ_TEMPLATE.md`](../squad/19-SCREENS_RFQ_TEMPLATE.md) or [`../squad/REMAINING-SCREENS_RFQ_TEMPLATE.md`](../squad/REMAINING-SCREENS_RFQ_TEMPLATE.md).

Separately, [`../squad/8-MODULES_FIRST_PRODUCTION_SLICES.tsv`](../squad/8-MODULES_FIRST_PRODUCTION_SLICES.tsv) (Legal, Skill Lab, Case Practice, Interview, Coach, Warmup, Negotiation, Job Search) uses the same rule: paths are **targets** until created from [`../squad/8-MODULES_RFQ_TEMPLATE.md`](../squad/8-MODULES_RFQ_TEMPLATE.md). Index: [`../features/8-modules-production-ready-pack-v1.md`](../features/8-modules-production-ready-pack-v1.md).

[`../squad/REMAINING-SCREENS_FIRST_PRODUCTION_SLICES.tsv`](../squad/REMAINING-SCREENS_FIRST_PRODUCTION_SLICES.tsv) lists further screen slices (Dashboard, Profile, Applications, …); create RFQs from [`../squad/REMAINING-SCREENS_RFQ_TEMPLATE.md`](../squad/REMAINING-SCREENS_RFQ_TEMPLATE.md). Index: [`../features/remaining-screens-production-ready-pack-v1.md`](../features/remaining-screens-production-ready-pack-v1.md).
